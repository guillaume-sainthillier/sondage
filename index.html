Récupération

Si vous possédez Git, tapez dans un invité de commande:

$ cd path/to/any/folder
$ git clone https://github.com/guillaume-sainthillier/malp.git

Si vous ne possédez pas Git (ce qui est déconseillé), téléchargez le projet au format Zip ou Tar.gz, et extrayez le dans n'importe quel dossier de votre choix.

Déploiement

Dans un invite de commande, tapez:

$ cd path/to/dnc/Client
$ java -jar DNC_Client.jar

Pré requis

Vous devez :

  • Posséder Java 6+
  • pour Windows uniquement : Ajouter les exécutables de Java dans votre PATH

En fait, reportez vous aux installations de :

  • http://www.java.com/fr/download/
  • pour Windows uniquement : http://www.java.com/fr/download/help/path.xml

Pour de plus amples informations, veuillez trouver le Manuel Utilisateur au format PDF à la racine du projet.
